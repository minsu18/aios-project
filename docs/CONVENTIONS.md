# AIOS Project Conventions

## Naming

- **Project name**: Use **AIOS** (one word, no hyphen or space).
- **Avoid**: "AI-OS", "AI OS", "IOS" — to prevent confusion with Apple's iOS.
- **Full form** (when expanded): "AI-Native Operating System", but the product name is **AIOS**.

## Language

- **All project outputs**: English
  - Code comments
  - README, docs, architecture
  - Commit messages
  - Issue/PR titles and descriptions
- **Rationale**: Global open-source project; English ensures worldwide contributors can understand and contribute.
