# 기여 가이드

AIOS 프로젝트에 기여해 주셔서 감사합니다.

## 개발 환경

- Rust (nightly) — 커널, HAL, 드라이버
- Python 3.10+ — AI 코어 프로토타입 (선택)

## 브랜치

- `main` — 안정
- `develop` — 개발
- `feature/*` — 기능 작업

## Pull Request

1. 이슈를 먼저 열거나 기존 이슈에 연관
2. `develop`에서 `feature/xxx` 브랜치 생성
3. 변경 후 PR 생성
4. 리뷰 후 머지

## 커밋 메시지

- `feat:`
- `fix:`
- `docs:`
- `refactor:`
