# Contributing

Thank you for contributing to AIOS.

## Development

- Rust (nightly) — kernel, HAL, drivers
- Python 3.10+ — AI core prototype (optional)

## Branches

- `main` — stable
- `develop` — development
- `feature/*` — feature work

## Pull Request

1. Open or link an issue
2. Create `feature/xxx` from `develop`
3. Submit PR
4. Merge after review

## Commit Messages

- `feat:`
- `fix:`
- `docs:`
- `refactor:`
